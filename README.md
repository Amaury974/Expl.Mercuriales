<h2>Expl.Mercuriales</h2>
Exploration de l'évolution des prix des denrées alimentaires agricoles produites et venudues à La Réunion. 
Basé sur les mercuriales produites par DAAF (données publiques).

<h4>Utilisation :</h4>
<ol> 
  <li>exécuter installation packages.R</li> 
  <li>modifier Run.bat pour flécher l'emplacement de R sur l'ordinateur
  <li>exécuter Run.bat</li>
</ol>
Vous pouvez aussi utiliser l'icône pour créer un joli raccourci.
